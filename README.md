# SQL Editor

Setup test environment and open page [http://localhost:4450/sql.html](http://localhost:4450/sql.html)

```
lein less auto
lein frontend
lein example-datasource
lein server
```

Check versions
```
lein ancient
```

Display dependencies
```
lein deps :tree
```

Check style
```
lein kibit
```

Format code
```
lein cljfmt fix
```
